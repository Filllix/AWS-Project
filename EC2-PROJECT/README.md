# EC2 Project

This section contains AWS EC2-related project documentation. The projects demonstrate how to deploy a web server, improve availability with load balancing and auto scaling, monitor infrastructure, and manage basic identity access.

## Projects

| No. | Project | Description |
| --- | --- | --- |
| 1 | [Simple Web Deployment on AWS EC2](../docs/ec2/simple-web-deployment-on-aws-ec2.pdf) | Deploy a simple website on an EC2 instance using Linux and Nginx. |
| 2 | [Application Load Balancer](../docs/ec2/create-application-load-balancer.pdf) | Configure an ALB to distribute HTTP traffic to EC2 targets. |
| 3 | [Auto Scaling Group](../docs/ec2/create-auto-scaling-group.pdf) | Create an Auto Scaling Group to improve availability and scalability. |
| 4 | [CloudWatch Monitoring](../docs/ec2/create-cloudwatch-monitoring.pdf) | Configure basic monitoring using Amazon CloudWatch. |
| 5 | [IAM User](../docs/ec2/create-iam-user.pdf) | Create an IAM user and understand access management basics. |

## AWS Services Used

- Amazon EC2
- Amazon VPC
- Security Groups
- Application Load Balancer
- Auto Scaling Group
- Amazon CloudWatch
- AWS IAM

## Learning Outcomes

- Provision and access EC2 instances.
- Deploy a basic Nginx web server.
- Configure traffic routing with an Application Load Balancer.
- Improve reliability using Auto Scaling.
- Monitor infrastructure with CloudWatch.
- Apply basic IAM access control practices.
