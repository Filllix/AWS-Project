# Terraform Project

This section contains Terraform-focused AWS documentation. The projects introduce Infrastructure as Code concepts and show how Terraform can be used to provision AWS infrastructure in a repeatable way.

## Projects

| No. | Project | Description |
| --- | --- | --- |
| 1 | [Deploy an EC2 Server Using Terraform](../docs/terraform/deploy-ec2-server-using-terraform.pdf) | Provision an AWS EC2 server using Terraform configuration. |
| 2 | [Create a Terraform Module](../docs/terraform/create-terraform-module.pdf) | Structure reusable Terraform code with modules. |

## Tools and Services

- Terraform
- AWS Provider for Terraform
- Amazon EC2
- AWS IAM

## Learning Outcomes

- Understand Terraform project structure.
- Initialize and apply Terraform configuration.
- Provision AWS resources with Infrastructure as Code.
- Organize reusable infrastructure logic with modules.
- Build a foundation for DevOps automation workflows.
